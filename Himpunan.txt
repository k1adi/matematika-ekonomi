1. Himpunan ditulis dengan huruf KAPITAL (A, B, C, D...)

2. Anggota himpunan ditulis dengan huruf kecil (a, b, c, d...)

3. Contoh penulisan himpunan nama nama buah
   => A = {salak, manggis, apel, jeruk}

4. "|" dibaca sedemikian rupa sehingga
   => B = {2, 4, 6, 8, 10} atau B = {x | x = bilangan genap}
   => B yang beranggotakan x sedemikian rupa sehingga x adalah bilangan genap
   
5. "∈" Jika suatu benda / objek merupakan anggota dari himpunan.
   => T = {a, i, u, e, o} atau T = {x | x = huruf vokal}
   => a ∈ T = a anggota himpunan T
   => i ∈ T = i anggota himpunan T
   => u ∈ T = u anggota himpunan T
   => e ∈ T = e anggota himpunan T
   => o ∈ T = o anggota himpunan T

6. "∉" Jika suatu benda / objek bukan merupakan anggota dari himpunan.
   => q ∉ T = q bukan anggota himpunan T
   => w ∉ T = w bukan anggota himpunan T
   => s ∉ T = s bukan anggota himpunan T
